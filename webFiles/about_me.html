More About Me

I have studied aerospace engineering but during my senior design project (NASA's University Student Launch Initiative) I became the payload electronics lead and discovered a true passion for computer engineering. I developed multiple PCB's and coded firmware for the first time, and I realized that this is where my true passion is. Since then I have enrolled at Louisiana State University to study computer engineering full time to turn my passion into a career. My skills have improved a lot since that first senior design project, and I have developed many things on my own including projects such as a new and improved flight computer and an embedded web server that is hosting this website!

I'm a very motivated worker on the projects that I am passionate about, and I greatly enjoy building projects from scratch. The best example of this is that when building my embedded web server, I implemented the HTTP stack from scratch using nothing but the RFC for HTTP 1.1 and Wireshark to examine the incoming packet structure and using the W5500 datasheet to implement SPI drivers. I find that I can learn a lot from reading documentation directly and I love finding my own path when designing projects and implementing things from the ground up.

I always have a new project or two (or five!) in the works, usually all having very different skills required, which lets me learn as much as possible and to diversify my interests. Currently I am designing a quadcopter drone (the flight computer and the electronic speed controllers) as well as having projects to expand on my knowledge of networking, including building an MQTT broker on an FPGA and building a single-board computer to run embedded Linux and function as a reverse proxy. Overall, my passion for embedded systems drives me forwards to learn as much as possible in many different fields.
